# misc stuff

#### Installing JDK
```bash
sudo apt-get install default-jre
sudo apt-get install default-jdk
```

#### Installing RPMs
```bash
sudo apt-get install alien
sudo alien my_package.rpm
sudo dpkg -i my_package.deb
```
